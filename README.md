# 📱 FitCity App Store Redirect

A lightweight HTML redirect page that automatically detects the user's device and redirects them to the appropriate app store.

## How It Works

- **Android** → [Google Play Store](https://play.google.com/store/apps/details?id=com.nextbrain.fit_city)
- **iOS (iPhone/iPad/iPod)** → [App Store](https://apps.apple.com/uz/app/fitcity-uz/id6752466966)
- **Desktop / Other** → Google Play Store (fallback)

## Usage

Host `index.html` on any static hosting service (GitHub Pages, Netlify, Vercel, etc.) and share the URL — users will be redirected automatically based on their device.

## Tech

Pure HTML + JavaScript. No dependencies, no build step.
